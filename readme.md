# Proxy Creator and Checker

- Writes the working proxies from the list to a txt file.

# free.proxy-list.py

## Usage

1. Install the necessary Python libraries:

```bash
pip install requests beautifulsoup4
```

2. Run the `free.proxy-list.py` file:

